AI Installation Protocol

Pass this instruction template along with the Skill URL to your AI agent. The agent will read this instructions, copy/vendor the files into your workspace, and hook it up to your discovery index.

Install this public skill repo into my workspace: <GitHub URL> Start from my workspace AGENTS.md or CLAUDE.md. Follow any WORKSPACE.md or skills/INDEX.md routing rules. Clone or vendor the repo under an appropriate project directory. Expose exactly one root skill to my global skill index or agent instructions. Keep private aliases, local paths, credentials, endpoint defaults, and business context in a local overlay, not in the public repo.
